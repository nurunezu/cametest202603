1. 液圧プレス用語（プレスの種類）

C形 / 門形 / コラム形

例：カニの片爪 vs 両腕

ロジック
フレームの「骨組み」の違い。C形は作業しやすいが、強く押すと口が開くように歪む。門形（SS）はガッチリしていて歪まず精密。

トランスファプレス

例：バケツリレー

ロジック
複数の工程を横に並べ、自動で材料を次へ送る仕組み。大量生産で、人手をかけずに複雑な形を一気に作るのに向く。

ダイスポッティング

例：職人の「合わせ鏡」

ロジック
上型が180度反転する。金型の最終調整（スポッティング）で、上型と下型の「当たり」を楽に、正確に確認するための専用機。

ストレッチフォーミング

例：シーツのしわ伸ばし

ロジック
板の両端を引っ張りながら型に押し付ける。航空機の外板など、大きな曲面をシワなくきれいに成形するための手法。

卓上形プレス

例：文房具のホッチキス

ロジック
机の上に乗るほど小型。電子部品の圧入や小さな金具の曲げなど、大きな力を必要としない繊細な作業に特化。

2. プレス機械の仕様用語（スペック）

オープンハイト / デーライト

例：本棚の棚の間隔

ロジック
「全開」にした時の隙間の高さ。この高さより背の高い金型は物理的に中に入れることができない。プレスの「胃袋」の大きさ。

ストローク長さ

例：パンチを振る距離

ロジック
スライドが上下に動ける距離。長いほど深い鍋やバケツを作れる。逆に足りないと、製品を金型から取り出す隙間がなくなる。

偏心荷重（へんしんかじゅう）

例：片足立ちでの踏みつけ

ロジック
スライドの真ん中から外れた位置に力がかかること。片減りやガイドの損傷、金型のズレの原因になり、機械の寿命を縮める。

慣性下降値 / 急停止時間

例：車の制動距離

ロジック
止まれ！と合図が出てから実際に止まるまでの距離。安全装置のセンサー位置を決めるための、命に関わる重要な数値。

3. プレス機械の操作用語（安全と運転）

両手操作式安全装置

例：二重の鍵

ロジック
両手で同時にボタンを押さないと動かない。両手がボタンの上にある＝「金型の中に手が入っていない」ことを物理的に保証する。

光線式安全装置

例：防犯レーザーセンサー

ロジック
目に見えない光のカーテン。手がそこを横切ると瞬時に機械を止める。物理的な扉がなくても「危険エリア」への侵入を防ぐ命綱。

寸動（インチング） / インチ運転

例：動画のコマ送り

ロジック
ボタンを押している間だけ「ミリ単位」で動かす。金型を合わせる（ダイセット）時、ガシャンといかないよう慎重に位置を確認する。

インタロック / キーロック

例：電子レンジの自動OFF

ロジック
「条件が揃わないと絶対動かない」仕組み。鍵がない、扉が開いている等の状態で勝手に動くのを防ぎ、他人の不用意な操作から守る。

足踏操作（フートスイッチ）

例：ミシンのペダル

ロジック
両手で材料を保持したまま運転できるが、その分リスクも高い。必ずガードや光線安全装置とセットで使用するのが鉄則。

4. 主要構成パーツと付属品（機械の構造）

ボルスタ / T形溝

例：精密なまな板

ロジック
金型を乗せる分厚い鉄板。 T形溝にボルトを差し込んで金型をガッチリ固定する。ここが少しでも歪むと製品にムラが出てしまう。

ダイクッション / ノックアウト

例：下から押し上げる座布団

ロジック
絞り中に材料に下から適度な圧力をかけてシワを防ぐ。終わったらピンで製品を突き落として金型から外す、プレスの「裏方」の主役。

タイロッド

例：建物を支える鉄筋

ロジック
分割されたフレームを一体化する巨大なネジ。 熱をかけて締め上げ、プレス中の強烈な反発力でフレームがバラバラにならないようガッチリ拘束する。

ギブ（スライドガイド）

例：引き出しのレール

ロジック
スライドがまっすぐ降りるように案内する板。 ここがすり減るとスライドがガタつき、金型の精度が落ちたり、刃先を破損させる原因になる。

安全ブロック

例：つっかえ棒

ロジック
金型の交換や点検の際、万が一油圧が抜けてスライドが落ちてきても挟まれないように物理的に支える、最後の砦。

5. 油圧用語（物理現象とトラブル）

キャビテーション / 空気混入

例：サイダーのしゅわしゅわ

ロジック
油の中に泡ができる。その泡が弾ける衝撃（超高圧）は鉄を削り取るほど凄まじく、ポンプをガリガリに削り破壊してしまう。

サージ圧 / デコンプレッション

例：炭酸を急に開ける「プシュッ」

ロジック
高圧の油を一気に抜くと衝撃（サージ）で配管が壊れる。 上がり始めに「ひと呼吸(デコンプ)」置いてゆっくり圧を抜くのが、機械に優しい動き。

ドレン / 漏れ

例：おねしょ / 排水路

ロジック
隙間から逃げた油の帰り道。本来の仕事をする油が漏れすぎるとパワーが出なくなり、漏れた場所は熱を持つので故障の前兆になる。

6. 油圧の制御とバルブ（機械の司令塔）

リリーフ弁 / 減圧弁 / アンロード

例：ヤカンの笛 / 蛇口の絞り

ロジック
圧力が上がりすぎたら「逃がす(リリーフ)」、必要な分だけ「下げる(減圧)」。 仕事がない時にポンプを休ませるのがアンロード。省エネと爆発防止の要。

プレフィル弁（満油弁）

例：ジョッキへの「ドバドバ」注ぎ

ロジック
ラムが高速で降りる時、ポンプでは油が追いつかない。 頭上の巨大タンクから重力で一気に油を吸い込ませるための、巨大な「ごくごく」飲口。

カウンタバランス弁

例：おもりの自重支え

ロジック
何トンもあるスライドが、自重で勝手にドスンと落ちてこないように下から支える力。 これがないと機械が自重で暴走してしまいます。

シーケンス弁

例：順番待ちのゲート

ロジック
「Aの動きが終わるまで、Bは動くな」という順序を守らせる。 クランプ（固定）が完了してから加圧を開始する、といった「段取り」の守り神。

アキュムレータ

例：油圧の「充電式電池」

ロジック
油の圧力を溜めておく。 瞬間的に巨大なパワーが必要な時や、ポンプが止まった時のバックアップとして機能する「貯金箱」。
